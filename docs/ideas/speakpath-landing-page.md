# SpeakPath — Landing Page

**SpeakPath** is the public brand for the AgentVoice app. The landing page is a static marketing site with no auth or backend — pure content.

This proposal is split into two documents:

- **[speakpath-landing-content.md](speakpath-landing-content.md)** — All copy, section structure, and design direction. Ready to hand off to a designer or drop into any site builder.

- **[speakpath-landing-architecture.md](speakpath-landing-architecture.md)** — Tech stack, repo structure, DNS setup, and deploy steps. The technical blueprint for when you're ready to build.
