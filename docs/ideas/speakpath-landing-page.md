# SpeakPath — Landing Page

## Summary

Create a static marketing landing page at speakpath.dev for the SpeakPath brand. The page
is fully decoupled from the app (separate repo, Netlify deploy) and serves as the public
entry point that converts visitors into app users. Covers copy/content and the technical
build-and-deploy plan.
